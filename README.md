# socialApp

## Description

This is a social application built using Node.js, Express, and MongoDB. It allows users to create, read, update, and delete posts and comments.

## Features

- Create, read, update, and delete posts
- Comment on posts
- Environment variable support using dotenv

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yoyoSL/socialApp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd socialApp
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

- To start the application in production mode:
  ```bash
  npm start
  ```
- To start the application in development mode:
  ```bash
  npm run dev
  ```
